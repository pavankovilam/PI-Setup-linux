Automated Setup For Pre-installed OS

Download AnyDesk & TeamViewer if not Downloaded Previously

cd ~/Downloads
rm -r ./*
wget https://download.anydesk.com/rpi/anydesk_7.1.3-1_arm64.deb
wget https://download.teamviewer.com/download/linux/teamviewer_arm64.deb

Install Anydesk & TeamViewer


cd ~/Downloads
sudo apt install ./*.deb
 echo "nkss@123" | sudo anydesk --set-password
sudo teamviewer license accept
teamviewer
